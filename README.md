# NordWeb - Professional Web Development Services

A professional portfolio and business website for an 18-year-old engaged web developer who specializes in building and fixing websites for companies and individuals.

## About

This website showcases professional web development services including:
- Custom website development from scratch
- Website repair and bug fixes
- Responsive design for all devices
- Performance optimization
- Website redesign services
- Business website solutions

## Features

- **Modern Design**: Clean, professional interface with smooth animations
- **Responsive Layout**: Works perfectly on desktop, tablet, and mobile devices
- **Service Showcase**: Clear presentation of available services
- **Contact Form**: Easy way for potential clients to get in touch
- **Professional Appeal**: Designed to attract both business and individual clients

## Getting Started

To view the website, simply open `index.html` in your web browser.

```bash
# Clone the repository
git clone https://github.com/NordWeb-del/NordWeb.git

# Navigate to the directory
cd NordWeb

# Open index.html in your browser
# On macOS: open index.html
# On Linux: xdg-open index.html
# On Windows: start index.html
```

## Structure

- `index.html` - Main website file with all content sections
- `styles.css` - All styling and responsive design rules
- `README.md` - This file

## Sections

1. **Hero** - Eye-catching introduction with call-to-action
2. **About** - Personal introduction and background
3. **Services** - Six key service offerings
4. **Why Choose Me** - Benefits and value propositions
5. **Contact** - Contact information and form

## Technologies Used

- HTML5
- CSS3
- Responsive Design
- CSS Grid & Flexbox
- CSS Animations

## Customization

To customize the website:
1. Update personal information in `index.html`
2. Modify colors in `styles.css` (see CSS variables in `:root`)
3. Add or remove services by editing the services section
4. Update contact information in the contact section
